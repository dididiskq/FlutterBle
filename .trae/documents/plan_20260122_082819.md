# 修改iOS应用名称与Android保持一致

## 需求分析
- Android应用名称："Ultra bms"（从AndroidManifest.xml第25行获取）
- iOS应用当前名称："Untitled1"（CFBundleDisplayName）和"untitled1"（CFBundleName）
- 需要将iOS应用名称修改为"Ultra bms"，与Android保持一致

## 实现步骤

### 1. 修改iOS Info.plist文件
- 修改 `CFBundleDisplayName` 字段，将其值从 "Untitled1" 改为 "Ultra bms"
- 修改 `CFBundleName` 字段，将其值从 "untitled1" 改为 "Ultra bms"

### 2. 验证修改
- 运行 `flutter build ios --debug` 命令，确保修改后的配置文件可以正常构建

## 具体修改内容

**文件**：`d:\pro\vscodePro\FlutterPro\untitled1\ios\Runner\Info.plist`

**修改前**：
```xml
<key>CFBundleDisplayName</key>
<string>Untitled1</string>
<key>CFBundleName</key>
<string>untitled1</string>
```

**修改后**：
```xml
<key>CFBundleDisplayName</key>
<string>Ultra bms</string>
<key>CFBundleName</key>
<string>Ultra bms</string>
```

这个修改将确保iOS应用在主屏幕上显示的名称以及内部使用的Bundle名称都与Android应用保持一致。
